# Basic E-Commerce Site 🛒

A lightweight, responsive front-end e-commerce web application built entirely with vanilla web technologies. This project serves as a foundational digital storefront, designed to cleanly showcase inventory—such as live aquarium fish, premium fish food, and aquatic supplies—and provide a seamless shopping experience for customers.

## 🛠️ Tech Stack
* **HTML5:** Semantic structuring across multiple pages (`index`, `shop`, `cart`, `contact`).
* **CSS3:** Custom styling and responsive layouts via `styles.css`.
* **Vanilla JavaScript:** Client-side dynamic logic, cart state management, and user interactions handled in `app.js`.
